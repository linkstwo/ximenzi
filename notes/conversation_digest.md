# 对话整理摘要（技术版）

本文件是根据项目讨论整理的技术摘要，不是原始逐字聊天记录。为避免泄露隐私、账号、手机号、密钥或比赛敏感原文，只记录与 PCS7/SMPT1000 控制设计直接相关的内容。

## 1. 项目定位

当前项目是 CIMC“西门子杯”智能制造工程设计与应用类赛项：流程行业自动化方向。工艺对象主要围绕 SMPT1000 平台中的自然循环锅炉，控制系统使用 Siemens PCS7。

核心目标不是只把程序照抄上去，而是要理解：

- 变量地址和阀门作用；
- CFC 页之间的信号关系；
- SFC 开车顺序；
- PID 自动/手动切换逻辑；
- 产量、成本、安全、稳定性四者之间的折中。

## 2. 已讨论过的关键问题

### 2.1 串级控制投运

串级控制通常需要先把副环调顺，再投主环。副环负责快速跟随执行量，例如流量或阀门相关回路；主环负责慢变量，例如温度、压力或液位。调试时不要一上来就主副环全自动，否则外环输出变化会把内环带崩。

### 2.2 锅炉水位三冲量/三向量思路

汽包水位不能只看 LI1102。高负荷时蒸汽流量变化会引起虚假水位，给水流量也需要作为前馈/反馈信号参与。实际程序中要看：

- LI1102：汽包水位；
- FI1101：汽包上水流量；
- FI1105：过热蒸汽流量；
- FV1101：汽包上水调节阀。

### 2.3 SP_EXT 外部设定

SP_EXT 不是随便接的常数，它常常来自 SFC 或上级控制逻辑。开车阶段 SFC 会给某些 PID 一个阶段性设定值，例如水位目标、压力目标、负荷目标。CFC 中某些 ADD/SUB/MUL 块看起来奇怪，可能是在把 SFC 设定、前馈量、补偿量叠加后送到 PID 或阀门。

### 2.4 手动初始开度与自动切换

开车时很多阀门不能一开始就自动。更合理的方式是：

1. 先手动给一个安全、不会冲击过大的初始开度；
2. 等 PV 接近 SP，趋势稳定；
3. 再切自动；
4. 串级时先副环自动，再主环自动。

如果手动开度给太大，PV 直接冲过目标，PID 再自动会产生大反向动作，导致产量、温度或压力波动。

### 2.5 通风、燃油、点火的先后

正常锅炉逻辑应先具备安全通风条件，再进燃油和点火。通风和燃油不是完全无脑同时，而是应先启动风机/打开烟道挡板，形成基本通风，再打开燃油管线和燃油泵，最后点火。这样更符合炉膛安全逻辑。

## 3. 重点变量关系

### 3.1 主要 PV

- FI1101：汽包上水流量
- FI1102：去减温器的汽包上水流量
- FI1103：燃油流量
- FI1104：进风量
- FI1105：过热蒸汽流量
- LI1102：上汽包水位
- TI1104：过热蒸汽温度
- PI1102：炉膛压力/真空度
- PI1103：汽包压力
- PI1104：过热蒸汽压力
- AI1101：烟气含氧量

### 3.2 主要 MV

- FV1101：汽包上水流量调节阀
- FV1102/FV1103：过热蒸汽温度调节阀 A/B
- FV1104：燃油流量调节阀
- S1101：变频风机转速调节
- FV1105：过热蒸汽流量调节阀
- DO1101：烟道挡板
- HV1101：汽包上水管线旁路阀
- HV1102：去蒸汽管网的管道阀

## 4. 开车逻辑简表

1. 初始：关闭调节阀、机泵、工艺阀门。
2. 打开放空：打开汽包顶部放空阀。
3. 上水：打开上水截断阀和相关调节阀，启动上水泵，使汽包水位接近 50%。
4. 通风/燃烧：打开烟道挡板，启动鼓风机，建立通风；再打开燃油调节阀和燃油泵，点火升压。
5. 升压后出汽：汽包压力到 1.5MPa 以上后，关闭放空，打开过热蒸汽出口相关阀门，控制温度、压力、流量和水位。
6. 提负荷：调节出料/蒸汽量，使 FI1105 进入指定产量范围，同时兼顾燃油量、烟气含氧量和安全约束。

## 5. 比赛侧注意

- 初赛现场上机时间有限，基础连接、IP、CEthernet、下载问题要提前排除。
- PCS7 工程比赛后需要归档提交，存在查重要求。
- 程序不能只靠抄，PID 参数和投运顺序非常关键。
- 正式赛题和样题可能在物性、参数、设备特性、控制要求和扰动类型上有差异。
