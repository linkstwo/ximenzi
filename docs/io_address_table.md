## 3. 使用提醒

- PV 是从 SMPT1000 读回来的测量值，不要当成阀门输出写。
- MV 是写给 SMPT1000 的操作量，DO 多用于开关阀/泵/风机/点火，AO 多用于连续阀门开度或转速。
- CFC 内部命名建议统一使用位号，例如 `PV_FI1105`、`MV_FV1105`，避免 FI/FV 混淆。
