# CIMC 西门子杯流程行业自动化（PCS7 + SMPT1000）项目资料库

本仓库用于保存和整理 **CIMC“西门子杯”流程行业自动化方向** 的项目资料、控制方案分析、PCS7/CFC/SFC 调试笔记，以及与 SMPT1000 工业锅炉对象相关的学习记录。

> 注意：本仓库不保存原始聊天逐字稿、账号密钥、手机号、报名材料等敏感信息；只保留经过整理的技术摘要与项目笔记。

## 当前项目主线

当前对象是 **SMPT1000 自然循环锅炉 / 过热蒸汽系统**，控制平台为 **Siemens PCS7**。重点不是单纯抄程序，而是理解并能调通：

1. 开车 SFC 顺序：上水、通风、燃油、点火、升压、并汽/出汽、提负荷。
2. CFC 控制回路：汽包水位、燃油/风量、炉膛压力、过热蒸汽温度、过热蒸汽压力/流量。
3. PID 投运顺序：手动初始开度、自动切换、串级主副环顺序、SP_EXT 外部设定来源。
4. 评分相关指标：产量、成本/燃油量、烟气含氧量、安全约束、稳定性。
